the HFLOAT package

The HUGE-FLOAT package It's for computations with very long (huge!) floating point numbers.
With hfloat you can compute pi to a million digits.
New HFLOAT version (15.JUN.1999): hf990615.tgz, now distributed under the GPL.
Please read whatfor.txt and hfloat.lsm
Please note that the doc is currently not up to date.

How to Compute pi & e

  • See the text included in my hfloat package !
  • The shortest programs for pi and e you have ever seen: calc-pi.c and calc-e.c.
  • These the so called spigot algorithm. two remarks about the pi spigot algorithm: spigot.txt
  • The spigot algorithm for pi, a bit more explicit: spigotpi.c
  • Christoph Haenel gives comments and improved versions of the code: spigot.haenel.txt
  • Texts on algorithms for the computation of pi: calc-pi.txt and calc-pi2.txt
  • An interesting link to the CECM: http://www.cecm.sfu.ca/organics/papers/.
  • A DOS executable (in uuencoded form) that computes almost 10000 digits of pi, 125 bytes long: pi.com.uu. Included is the commented disassembly by Greg Smith.
  • Compute the millionth hex digit of pi without computing all previous digits: C++ code, prints also binary and saves some table look up: pihex.cc.
  • And of course you have to get the book 'Pi, Algorithmen, Computer, Arithmetik' by Jörg Arndt and Christoph Haenel, it will make your life better.
