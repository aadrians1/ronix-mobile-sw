Wintesla/Firmwareupdate

Was ist ein Firmwareupdate?

Mit der Nokia Service Software Wintesla, die zur Wartung und Pflege des Handys (Nokia 6110, 6150 u.s.w.) eingesetzt wird und nur in Nokia Servicecentern zur Geltung kommt, kann man von zuhause aus so ziemlich alles an seinem Handy verändern: Security Code,  Menuestruktur...

Man benötigt:

1. Wintesla 

2. Firmwarefiles                             

ACHTUNG:

Ein Firmwareupdate ist mit Wintesla nicht möglich und wird NIE möglich sein! Flashbox FPS4 und Security Box wird man NIE bekommen! Ohne die beiden Geräte ist ein Firmwareupdate nicht möglich!
Außerdem benötigt man ein MBUS Kabel.

ATTENTION:

THERE IS NO WAY TO MAKE A FIRMWAREUPDATE WITHOUT THESE TWO BOXES!

!! Hier könnt ihr günstig Nokia Data Suite kompatible Datenkabel kaufen !!

Die Bedienung von Wintesla ist einfach. Sollte es dennoch Probleme bei der Bedinung  geben schickt Ihr einfach eine Mail an mich.
