Термины и сокращения, используеме в сотовой связи
Релиз 1.1
Сокращение Термин (анг/русский) Пояснение
BS Base Station, Базовая станция Приемопередающее, управляющее и коммуникационное оборудование, составляющее сеть, состоит из BTS+BSC
BSC Base Station Controller, Контроллер базовой станции Аппаратура управления базовыми станциями
BTS Base Transciever Station, Приемопередатчик базовой станции Собственно приемная и передающая аппаратура
CDMA Code Division Multiple Access, Множественный доступ с кодовым разделением Способ использования радиочастот, при котором абоненты разделяются методом специального кодирования
DSC Digital Cellular System, цифровая сотовая система Достаточно общий термин, часто ассоциируется с DSC-1800
DECT Digital Europian Cordless Telecommunication Стандарт для бытовых телефонов. В последнее время стали появляться аппараты поддерживающие DECT наряду с GSM (известен аппарат от Sagem 900/1800/DECT).
EMC Equipment Manufacturer Code Код производителя оборудования
FDMA Frequency Division Multiple Access, Множественный доступ с  разделением по частотам Способ использования радиочастот, когда в одном частотном слоте находится только один абонент, разные абоненты используют разные частоты в пределах соты.
HS Hand Set, ручной телефон, "трубка" Оконечное оборудование сотовой сети
IMEI International Mobile Equipment Identity Уникальный международный идентификатор мобильного оборудования
IMSI International Mobile Subscriber Identity Уникальный международный идентификатор абонента
MNC Mobile network Code Код сети мобильной связи
PCS Personal Communications Service Служба персональной связи, часто ассоциируется с PCS-1900
PSN Personal Communications Network Сеть, обеспечивающая сервис персональной связи
PIN, PIN2 Personal Identification Number Секретный код, служащий для ограничения доступа к оконечному сотовому оборудованию (трубке)
SIM Subscriber Identity Module СИМ-карта, модуль, идентифицирующий абонента сотовой сети. В настоящее время используется только в различных можификациях GSM.
SMS Short Messsage Service Сервис коротких сообщений
TDMA Time Division Multiple Access, Множественный доступ с   разделением по времени Способ использования радиочастот, когда в одном растотном слоте находится несколько абонентов, разные абоненты используют разные временнЫе слоты для передачи. В GSM один частотный слот делится на 8 временнЫх.
