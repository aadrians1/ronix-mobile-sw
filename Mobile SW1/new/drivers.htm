Download Tips

Before you update a driver, please check our Hotline Tips (FAQ).
You'd like to save driver packages to floppy, however, the files are to big?
Use our tool SPLIT that splitts big files into smaller files, you may define the size of these parts. Starting the first splitted part the complete file will be created (no additiona tool is required). Download SPLIT.EXE.
To avoid unnecessary downloads: for this we have our  UPDATE WIZARD, a tool which will check your installed DIAMOND hardware and driver version, compares it with the driver on our Website (Internet access is required) and updates it automatically.
For older products which are not supported from this tool please check the correct name of your DIAMOND hardware to make sure to download the correct driver.
What means "Installation starts automatically" in the file description? The update packages for our newer products are already installation packages. That means after you double-click on the package.EXE an installation programm will be started and installs the driver automatically.
What means "SFX file" in the file description? Insofar as the installation starts not automatically all our packages are self-extracting files - SFX files. That means after you double-click on the package.EXE a menue will appear which shows you the target folder and allows you to change this folder. After you have extracted this package you will find a README with further informations about the installation of the driver.
Your connection is to slow? You'd like to have the driver on CD? Just contact Upgrade Options!
Windows NT 4.0 drivers for older Diamond products? This drivers are already included in Windows NT 4.0. You will find them under Control Panel - Display - Settings
