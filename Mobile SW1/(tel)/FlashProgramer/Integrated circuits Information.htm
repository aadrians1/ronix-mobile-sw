Flash / EEPROM Programmer, Rev C

Here's a build-your-own programmer & V1.4 software for those on a limited budget. I used these design goals in order to come up with this circuit. It can only program 5V only memory so it can't program EPROMs or some EEPROMS. It's designed to be simple & inexpensive rather than a program-everything circuit. Why use EPROMs anyway? You need special equipment to erase them and it takes 20 minutes to do it.
