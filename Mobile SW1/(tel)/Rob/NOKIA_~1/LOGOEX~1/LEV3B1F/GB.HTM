LogoExpress

1.功能:

    - 修改 Nokia 61xx、51xx 开机图标, 系统图标与来电图标
    - 浏览与预视目录中所有的图标文件
    - 支持 .nol、 .ngg、 .bmp、.jpeg logo 图标文件格式.
    - 需要 DAU-9P（F-BUS cable），不需 Nokia Cellular Data Suite（NCDS）
    - 直接设安图标不需要经由简讯（61xx）
    - 系统商选项可经由文字文件自定
    - 将简讯存在 SIM 卡上时可选择存在“已收讯息（Inbox）” 或“已存讯息（Outbox）”
    - 简单之图标编缉功能
    - 完整Nokia 图标修改功能而且完全免费.

2. 安装

   将下载之 zip 档解开到一暂存目录，执行 Setup 并遵照安装指示。

3. 使用

    1）. 主要功能都可从下拉式选单或工具列选择，所有工具列接钮都有工具提示，左下方之磁盘
   与目录窗口可用来浏览与选择目录，选择的目录之所有图标可在右下方的预窗口浏览，点选预窗
   口的图标可加载该图标以编缉或加载手机
   2）.将图标加载手机需要 Nokia DAU-9P（F-Bus）连接线。首先以连接线连接手机与计算机，
   并在 Setup->Properties 中设定计算机使用的 com port，若要使用简讯传送的功能必须先
   设定 SMSC（简讯中心）号码，若你不知道简讯中心号码可直接按 "Read from phone" 从你
   的手机读取或在手机上按讯息->讯息设定->Set 1-> 讯息中心号（1-4-1-1 for 61xx、2-4-1-1
    for 51xx）。
   3）在 Setup -> Properties 中的 Operator list file 请选择 operatorlist.txt，可自行设定
   系统商列表，此档案可自行编缉，格式是 tab 分隔的纯文字文件。
    4）将图标加载手机有 3 种方法：（1）直接设定，只对 61xx 有用；（2）先存到SIM 卡上，
   再以操作-> 转送将图标传给自己手机或别人的手机号码。这个功能可让你将多个图标事先存在
   SIM 卡，不需计算机即可更换图标；（3）直接以简讯传送, 需先设定简中心号码，然后输入要
   传送的手机号码。方法（2）和（3）需要简讯中心直援 8bit 简讯。
