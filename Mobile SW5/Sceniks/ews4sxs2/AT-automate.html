It is possible to set up a recurring indexing process using the cron facility, which allows you to run a command at specific times. To invoke an indexing process, use the aindex.pl script, which can be found in the directory in which you installed Excite for Web Servers.

The usage for aindex.pl is as follows: 

    aindex.pl [collection name]

So, for example, if you wanted to index the collection 'webdata' every night at midnight, and your software was installed in /usr/local/etc/architext, you would need to add the following line to your crontab file using the crontab command: 

    0 0 * * * /usr/local/etc/architext/aindex.pl webdata
Be sure that the cron job will be run under a user id that has read and write privileges to the directory that your indexes are stored in.

For more detailed information on the use of cron and crontab, check your system's man pages using the man command.
