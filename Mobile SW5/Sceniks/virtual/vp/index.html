Eight 8-Bit PWM's This example code provides 8 PWM outputs on the PORT B pins. Each PWM is individually adjustable (through code modifications), for different duty cycles. This should be used to toggle LED's or drive small speakers directly off the PORT B pins, although frequencies of up to 1MHz are available. pwms.src
8-Bit ADC
 The traditional 8-Bit A/D Converter relies on comparison with a known reference, through an amplifier, to adjust for the difference in the unknown signal. With some software tricks, we can implement a simple 8-bit A/D converter on any CMOS input to the SX device. This is compatible with the SX Demo Board. adc_vp.src
