RINGS UNTIL ANSWERED

This changes the number of rings on your phone before diverting. To find your voicemail number call your voicemail, but stop the call before connection. Press redial once then press CLR once. This will show your voicemail number minus the last digit.. So to set the number of rings before diverting you would enter:

**61*+XXX(area code)XXXXXXX(dest. number)**30#@

This would change the ring time to 30 seconds.
N = 5-30 (seconds)

OPTION
CODE
SET number of rings
**61*YourVoiceMailNumber**N#@
CANCEL previous setting entered
##61#@
