Highly Integrated High-Performance Microcontrollers
The C167CR/SR are high-end members of the Infineon C166 family of full featured single-chip 16-bit microcontrollers. They combine high CPU performance with high peripheral functionality and enhanced I/O-capabilities. A wide variety of on-chip features such as large memories, multi-functional standard peripherals, and application-specific peripherals (e.g. CAN) is available. On-chip high-speed RAM and clock generation via PLL is provided. The C167CR features an on-chip CAN module which meets version 2.0B active of the Controller Area Network specification and which has been designed to fulfill the requirements of coming generations of automotive and industrial control applications.


 

C167CR Derivatives
  High-end 16-bit Microcontrollers with on-chip CAN-Module  
  Related Solutions/Applications

Features:
  • 16-bit CPU with 4-Stage Pipeline
  • CPU clock speed up to 33Mhz
  • 4kByte on-chip SRAM memory
  • Up to 128kByte on-chip ROM
  • One CAN module version 2.0B active (only C167CR)
  • Peripheral Event Controller (PEC)
  • Two 16-channel Capture/Compare Units
  • 4-channel PWM Unit
  • 16-channel 10Bit A/D Converter
  • Two Multi-Funtional general purpose timer units with five 16-bit timers
  • Programmable Watchdog Timer
  • Two serial channels(synchronous/asynchronous and high-speed synchronous)
  • On-Chip Bootstrap Loader
  • Up to 111 General purpose I/O lines
  • 5 Volt operation
  • 144-Pin MQFP Package
  • Full Automotive Temperature Range: -400C to +1250C
