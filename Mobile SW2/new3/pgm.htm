IC-Prog Prototype Programmer

 

Programs : 12C508, 16C84, 16F84, PIC 16F877, 24C16, 24C32, 93C46, 90S1200, 59C11, 89C2051, 89S53, 250x0, PIC, AVR , 80C51 etc.

This software package allows you to program all types of serial programmable Integrated Circuits.

I have added a toolbar and some other minor functions! Also the editor part has been updated (The EE part does not work yet!)

NEW!!: I have added "Through Pic Programming" for all smartcards with PIC+24C16. Just enable the Smartcard option in the Settings menu and choose the 24C16 as the device. Then use Read (F8) or Programm (F5) just as normally!!!! This can only be used with a serial port, because the communication is standard asynchronous communication (Phoenix). (Verify and Blank is NOW supported!!!!!!)

You can use Direct I/O or the Windows API to program a device, so you can use it with Windows NT. Windows API is only applicable to Serial Ports!
