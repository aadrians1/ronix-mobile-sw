Most Frequently Asked Questions:

(Q1) How long before I get it?
(A1) If it's in stock, then it still depends on where you live. All units are shipped from Hong Kong. If you are in Hong Kong, 1-2 days, Australia 4-8 days, USA/Canada 8-10 days (had been 4 weeks once), South Africa could be 3-5 weeks, and some South American countries including Chile could be 6 weeks, but normally I quote everyone 8 to 14 days. But Please, wait 3 weeks before asking about kits sent from HK. This is in the hands of the postal system. It is sent standard Air Mail, so there is no method of tracking the package.

(Q2) How much will it cost me?
(A2) The prices quoted are in USD, this is converted at the current exchange rate to AUD and rounded up.

(Q3) I have a 7407, or I was supplied a 7407 in the kit. How do I configure it to work?

(Q4) I live in Australia. Can I pop in and pick one up?
(A4) We have never seen a kit. These are shipped from Hong Kong. The problems of dealing with Australian Customs for simple imports has been resolved by not dealing with them. This becomes your problem, however we haven't heard of anyone being pulled up with customs on these items. Yes you can mail order from Australia, but verify it with an email message also.

(Q5) Will it program a PIC12C508/04/P, which is the Play Station MOD Chip?
(A5) Yes. You must get your own code. Heaps of this on the Internet. See my 508.html page for instance for Sydney mod chips, RIE, and OZ Chips.

(Q6) Is there a Schematic?
(A6) It is shown at:  http://www.picallw.com/

(Q7) Will these programmers read a PICmicro Chip?
(A7) Yes, providing it's not code protected. Most commercial applications you can bet, will have the code protection bit(s) set.

(Q8) Can I C.O.D. these products?
(A8) No, they are shipped from Hong Kong. We can't do C.O.D. from Hong Kong.

(Q9) I only want to program PIC16F84's. Do I have to register the software?
(A9) No, if you have a look at Don's Download Dungeon, you will find drivers for DOS, W9x, and NT for free.
Nigel Goodwin wrote these for DonTronics, and they are free to use for everyone. Both the registered and the free software will work with the DIY KIT-96, and the DonTronics DT001 board. Bojan has now available a free 84 burner aboard his P16PRO software.

(Q10) I can't get the new Stealth code to load correctly with P16PRO.
(A10) I don't answer any questions on playstation code, but if I did, I would tell you that when downloading any HEX file directly from net, you should edit the HEX file with an asci editor and erase last character in the line if it is different from
0,..9,A..F Webmasters should never put a raw hex file on the web, but a zip file.

(Q11) What are the power supply specs?
(A11) As written on the overlay of the kit 96 board: 16 to 30 VDC, 12 to 20VAC. Anything greater than 100ma is fine.
The DC input has a diode bridge, so polarity isn't a problem.

(Q12) I have built the unit and the LEDs don't seem to be doing much.
(A12) Grab the Windows P16PRO software which is linked from the P16PRO page. This will burn 84 type chips without registration, and has a large help file, including the setup parameters and test features. You have to configure your port and hardware, then you can test the LED operation.

(Q13) I can't figure out the holes for sockets on the KIT 96 board. What are the white lines?
(A13) The Aries type ZIF socket fits the 40 pin holes and does all chips. The P16PRO software tells you where to place the chip in the socket. The 8 and 18 pin positions are for smaller sockets if you wish to use them. The white lines are links.

P16PRO Software Registration:
You can Register P16PRO PICmicro software at DonTronics "on-line", using our web based secure Credit Card system for $20USD. The P16PRO software will work under DOS, W3.1 or W9x.
This software will allow you to use most "David Tait" variant programmers to program a very large number of PICmicro devices via the Parallel Printer Port of a PC. Follow the registration link to read the full story.

This is a kit of parts to use the P16PRO software of Bojan Dobaj to program all 8, 18, 28 & 40 pin DIP serial PICís - see the  listing below.

Parallel programmed PICís - 16C5X - are not supported by this programmer.

Do not confuse these programming methods with the serial port and parallel ports of a PC. A serial programmed PICís refers to the programming algorithm by which data enters the PIC. In this method the data bytes are entered serially onto 1 pin (like a shift register) and the 13V programming voltage is toggled onto a programming pin to latch and burn the byte. In the parallel programming method the whole byte is presented on an 8 pin port then the programming voltage is toggled.

You must download the latest version of the P16PRO software from:
http://www.picallw.com/

These are the PICís that this kit will program:
PIC12C5xx, PIC14000, PIC16C554, PIC16C556, PIC16C558, PIC16C61, PIC16C62, PIC16C62A, PIC16CR62,  PIC16C63, PIC16C64, PIC16C64A, PIC16CR64, PIC16C65, PIC16C65A, PIC16C66, PIC16C67, PIC16C620,  PIC16C621, PIC16C622, PIC16C710, PIC16C71, PIC16C711, PIC16C72, PIC16C73, PIC16C73A, PIC16C74,  PIC16C74A, PIC16C76, PIC16C77, PIC16F83, PIC16CR83, PIC16C84, PIC16F84, PIC16CR84,  PIC16C923, PIC16C924, PIC16C642, PIC16C662, and PIC16C715

New PICís can be added by entering them in the device.ini file of the software.

Schematic. You may download the schematic for the 40 pin hardware version from the website. There are several changes:

  • R9/680E should be labelled R11/680R
  • for C3 we have used 470uF/35V
We have made space on the board for C5/330pF from pin 6 of the 74LS05 to ground since there has been one only report that it may be required. We have not supplied this component.

Construction. There are 8 links to add to the board. We have supplied 8 18 28 & 40 pin IC sockets. However, for maximum flexibility in programming you may wish to supply your own 40 pin wide-slot ZIF socket. Use the 14 pin IC socket to mount the 74LS05. You will need to connect the programmer to the parallel port of a PC using a straight through cable from the on-board 25 pin PCB-mounted subd connector.
