***NEC DB 2000***

To see IMEI code type: *#06#
To Reset to defaults type: *73738# (send?)
To show Software version type : *#2820#

SP Lock info:
The procedure will show if your phone is locked. * # 3210 # (send?) for SIM lock info,
type : *#8140# (send?) for Net lock info.

SIM lock :
*#4960 # (send?) - Inquiry * 4960 * password * password # (send?) to Lock
#4960* password # (send?) to Unlock

[password] is an 8 digits number]

Net Lock:
*#7320# (send?) - Inquiry * 7320 * password * password # (send?) to Lock.
#7320* password # (send?) to Unlock.

[password] is an 8 digits number]

Net Lock 2:
*#2220# (send?) - Inquiry * 2220 * password * password # (send?) to Lock
#2220* password # (send?) to Unlock

[password] is an 8 digits number]

Subnet Lock:
*#1110# (send?) - Inquiry * 1110 * password * password # (send?) to Lock
#1110* password # (send?) to Unlock

[password] is an 8 digits number]
